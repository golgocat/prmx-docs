# PRMX V4 — Engineer Documentation

Curated reading set for engineers learning the PRMX V4 architecture and design.

Published via Mintlify at https://prmx.mintlify.app — start at [`docs/INDEX.md`](docs/INDEX.md).

## Local preview

```bash
npm install -g mintlify
mintlify dev
```

Opens `http://localhost:3000` with hot reload against `docs.json` and `docs/`.

## Layout

| Path | Purpose |
|------|---------|
| `docs.json` | Mintlify v4 navigation, theme, search config |
| `docs/INDEX.md` | Engineer reading guide; landing page |
| `docs/architecture/` | System overview, capital invariants, parametric insurance rulebook |
| `docs/hyperlane-migration/` | Cross-chain transport (post-cutover designs only) |
| `docs/product/` | App design, product catalog, Rainguard spec |
| `docs/guidelines/` | Test principles, UI design principles |
| `docs/operations/` | Restart guide, oracle workers reference |
| `logo/`, `favicon.svg` | Site brand assets |
